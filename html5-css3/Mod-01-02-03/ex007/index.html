Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>, que é não-semantica

Nesta frase, tenho um termo em destaque usando a tag <strong>(tem semântica).

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em>(tem semântica).

Evite usar tags não semântica, pois logo poderão deixar de funcionar.

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>. Porém, pode-se marcar um texto com CSS,
adicionando o atributo "style" e demais atributos do CSS.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno.

Porém, a tag <big> está obsoleta. Logo deixará de funcionar.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usa-se, para isso, a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.Usa-se, para isso, a tag <ins>, que é uma tag semântica. Atente-se para o caso da tag <u>, ela está obsoleta. O que significa que a aqualquer momento deixará de funcionar.

Texto Sobrescrito

Para inserir coisa do itpo x20+3.Usa-se para isso a tag <sup>.

Texto Subscrito

Para inserir coisa do tipo H2O.Usa-se para isso a tag <sub>.
