Listas de Definições

Para cirar listas de definições usamos a tag principal "<dl>", em seguida, as sub-tags: <dt> e <dd>, nesta seqüência.

HTML
linguagem de marcação para a criação de connteúdo de um site.
CSS
Linguagem de marcação para a criação de design de um site
JavaScript
Linguagem de programação para a criação de interatividade de um site.

Observação Importante:

O mecanismo de busca do Google prioriza páginas que usam estas listas de definições, quando alguém pesquisa por definições nele.
